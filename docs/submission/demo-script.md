# Demo Script

## 90-Second Script

`Today we’re showing ToolMarket Lite, a product that sells premium tool access one call at a time.`

`The problem is that subscriptions are a bad fit for agents and power users who only need one valuable action occasionally.`

`Here the buyer sees a premium web-fetch tool priced at 0.004 USDC per call.`

`When the buyer requests a result, the backend returns a 402-style payment requirement instead of silently billing in the background.`

`The buyer authorizes the payment, retries the request, and the tool executes immediately.`

`Now the result is visible, the seller dashboard updates, and the settlement reference is recorded.`

`This pricing model only works because transaction overhead is low enough to preserve margin at sub-cent prices.`

`That’s why Arc and Circle matter here: they make per-action, AI-native commerce practical instead of decorative.`
